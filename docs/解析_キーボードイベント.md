- モニタリング: monitorEvents(document, "keydown")
- 一覧: getEventListeners(document).keydown
- ブレークポイントを設置: ソース>イベントリスナーブレークポイント>キーボードを ON にする

# 特定のリスナーでキーを実行する

```js
getEventListeners(document).keydown[0].listener(
  new KeyboardEvent("keydown", {
    key: "i",
    code: "KeyI",
    keyCode: 73,
    bubbles: true,
    cancelable: true,
  })
);
```
