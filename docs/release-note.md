## インストール方法

```
- 【PC】 [Chrome ウェブストア](<https://chromewebstore.google.com/detail/mr-wplace/klbcmpogekmdckegggoapdjjlehonnej>)または[Edge アドオン](<https://microsoftedge.microsoft.com/addons/detail/mr-wplace/acdodonamhbokadiikkfnnliplijigip>)または[Firefoxアドオン](<https://addons.mozilla.org/ja/firefox/addon/mr-wplace>)からワンクリックでインストール
- 【iOS】 [Orion Browser by Kagi](<https://apps.apple.com/jp/app/orion-browser-by-kagi/id1484498200>) をインストール → 設定>詳細設定>「Chrome 拡張機能と Firefox 拡張機能」を ON → [Chrome ウェブストア](<https://chromewebstore.google.com/detail/mr-wplace/klbcmpogekmdckegggoapdjjlehonnej>)からインストール
- 【Android】 EdgeCanary アプリをインストール → 開発者モードにする(設定>「MicrosoftEdge について」> 1 番下のバージョンの数字を５回ぐらいタップ) → 戻って「開発者向けオプション」→「Extension install by id」→`acdodonamhbokadiikkfnnliplijigip` を入力
```

# 🧑‍🎨Mr. Wplace(v1.10.1)リリースノート

## 選択中の色のみを表示する機能

- 選択中の色を切り替えたとき、その色のみが表示されるように自動で切り替わります
- ペイント画面のパレットの上にあるボタンから利用できます
- いちいちカラーフィルター画面を開かなくて済むので快適になります

## 塗られていない色のみを表示する機能

- 背景/すでに塗った部分をグレーにして、塗ってない色のみが表示されます
- 巨大な絵などで、塗り残しが分からないときにすぐ見つけることができます
- 「選択中の色のみを表示する機能」と組み合わせることで別ゲーのようになります
- もちろん、このような視覚補助は wplace の規約で明示的に認められているので安心して利用できます

## ブックマークのタグ機能の強化

- ブックマーク画面の上部に一覧表示されるタグをクリックすると、そのタグのブックマークのみが表示されるようになります
- 既存のタグを編集できるようになりました。タグ名変更/色変更/削除ができます
- タグのエクスポートを強化しました。タグごとにブックマークをエクスポートができます
- ぜひお気に入りのタグをシェアしてみてください。他の人は、「インポート」から簡単にタグを読み込めます

## UI のダークテーマ対応

- ダークテーマに切り替えると、UI もダークテーマになります
- マップ右上のボタンから切り替えることができます
- マップも同時にダークテーマに切り替わります

## タイルのオフラインキャッシュの設定

- いくつまでのタイルをオフラインに保存するか設定できます
- データセーバーを ON にすると、ダウンロードしたタイルを利用するので、通信が抑えられ軽くなります
- 現在のストレージ使用量が表示されます
- 上限に達すると、最後にアクセスしたタイルから先に削除されます

## 画像編集でシャープネスに対応

- オプションでシャープネスを変更できるようになりました
- 輪郭線などを強調したいときに利用してください

## タイル一覧/ギャラリーの並び替え

- タイル一覧：「距離が近い順」「最近保存した順」「タイル数が多い順」「名前順」
- ギャラリー：「レイヤー順」「距離が近い順」「追加順」

## バグの修正/UI の調整

- 画像の進捗のカウントが合わない問題の修正
- モバイル環境で、画像編集画面のスクロールができない問題の修正
- 複数のタイルを同時に表示すると、しばしばエラーになる問題の修正
- 登録済みの画像を描画 OFF にするとプログレスバーなどが表示されなくなる問題の修正
- ドラッグアンドドロップでの画像変更がうまくいかない問題の修正
- リロードするなどすると進捗のカウントがリセットされる問題の修正

- マップクリックで表示される座標などの横にコピーアイコン設置
- UI をダークテーマに対応
- データセーバーモードでも、ペイントしたらタイルが更新されるようにしました
- シェア画面の UI を変更し、コピーしやすくしました

## その他

- カラーフィルター画面の UI を向上
- ブックマーク画面の UI を向上
- タイル一覧画面の UI を向上

- wplace の規約や、機能の完成度などに懸念点があるので封印しています。
